# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2025-11-26
### Added
- Windows Unicode path handling for creating/opening archives and manipulating files.
- Automated regression test covering Chinese directory and file names.

### Changed
- StormLib build script now targets Visual Studio 2022 and forces `/MT` runtime linkage for compatibility with node-gyp builds.
- Documentation updated with new repository location and release highlights.
